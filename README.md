# ToyProject-Gundam 🚀

간단하고 빠르게 구축한 로그인/로그아웃 기능 기반의 웹 애플리케이션입니다.  
Google OAuth를 활용해 사용자 인증을 구현하였고, **Cursor AI**, **Bolt.new**, **AWS**를 사용하여 실험적으로 개발한 토이 프로젝트입니다.

---

## 🛠️ 주요 기술 스택

- **Authentication**: Google OAuth 2.0 (로그인/로그아웃)
- **Frontend/UX**: Bolt.new (UI prototyping)
- **Backend / Logic**: Cursor AI 기반 자동화 개발 흐름 활용
- **Infra**: AWS (배포 및 리소스 호스팅)

---

## ✨ 프로젝트 목표

- Google 로그인을 통해 손쉽게 사용자 인증 구현
- Cursor AI 및 Bolt.new의 협업 기반 개발 워크플로우 실험
- AWS를 활용한 간단한 클라우드 배포 및 서비스 흐름 이해
- **작고 빠른 완성**, 그리고 **새로운 툴 실험**에 중점

---

## 🔐 기능

- ✅ Google 계정으로 로그인
- 🚪 Google 계정으로 로그아웃
- 🪄 Cursor AI를 통해 코드 업데이트 반복 실험
- 🌩️ AWS 상에 배포된 테스트 환경

---

## 🧪 테스트 및 배포

- 이 프로젝트는 **개인 실습 및 프로토타입 목적**으로 제작되었습니다.
- 정식 서비스용이 아니며, 기능은 간단히 동작만 확인할 수 있도록 구성되어 있습니다.
